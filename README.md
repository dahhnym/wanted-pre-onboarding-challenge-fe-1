# 원티드 프리온보딩 프론트엔드 1월 챌린지 사전과제

## Dependency

- React 18.2.0
- TypeScript 4.9.4
- React-router-dom
- Axios
- Styled-components

## 실행방법

```
$ git clone
$ npm install
// 서버실행
$ cd server
$ yarn start
// 클라이언트 실행
$ cd client
$ npm start
```
